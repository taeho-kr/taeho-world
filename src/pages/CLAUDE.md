# Pages 디렉토리

> 참고: 실제 라우팅은 `src/app/`에서 Next.js App Router로 처리.
> 이 디렉토리는 페이지별 전용 컴포넌트와 리소스를 코로케이션하는 용도.

## 페이지 구성

| 디렉토리 | 용도 |
|-----------|------|
| `HomePage/` | 랜딩, 히어로 섹션, 하이라이트 |
| `CareerPage/` | 경력 타임라인, 회사별 상세 |
| `InfoPage/` | 자기소개, 기술 스택, 연락처 |
| `MemoPage/` | 블로그/메모, MDX 콘텐츠 |

## 규칙

- 각 페이지 디렉토리에 `components/` 폴더를 두어 해당 페이지 전용 컴포넌트 관리
- 2개 이상 페이지에서 사용되면 `src/components/myUI/`로 승격
